# Channel: social — organic posts & threads (LinkedIn / X / Instagram)

Writes `spec/copy/social.md`. Ask the platform; if unspecified, default to **LinkedIn + X** (B2B SaaS).
Each post maps to a positioning pillar, objection, switching force, or proof point. Organic ≠ ad: lead
with value/insight, not a pitch.

## Formats by platform

- **LinkedIn:** 1,300–2,000 chars; first 2 lines are the hook (before "see more"). Formats: insight post,
  story post, how-to list, contrarian take. One idea per post; CTA soft (comment/DM), not "buy now".
- **X / Twitter:** single post ≤280 chars, or a **thread** (hook tweet → 4–8 body tweets → recap + CTA).
  Hook tweet must stand alone. Tech-audience, concise, no hashtag spam (0–2).
- **Instagram:** caption (hook in first line) + a **carousel** outline (cover → 4–6 slides → CTA slide).
  Visual-first; caption supports the image.

## Hook patterns (the first line earns the scroll-stop)

- Contrarian: "Most {audience} think {belief}. They're wrong."
- Result/story: "{Specific before→after, from positioning}."
- Pain callout: "If you've ever {pain}, this is for you."
- List promise: "{N} ways to {outcome} — without {objection}."

## Output structure (in the file)

Group by platform. Provide **3–5 posts per platform** (or 1 thread + 2 posts). Per item: the full copy,
a `— pillar/force:` annotation, and a soft CTA. For threads/carousels, number the parts. Keep brand voice
from positioning; no banned fluff; no hashtag stuffing.
